# Sistema de Controle de Desperdício de Merenda

Projeto desenvolvido como Trabalho de Conclusão de Curso (TCC) com o objetivo de registrar e analisar o desperdício de merenda escolar por meio de gráficos e armazenamento em banco de dados.

## Tecnologias utilizadas
- React
- JavaScript
- Java
- SQL
- HTML
- CSS

## Funcionalidades
- Cadastro de dados de desperdício
- Armazenamento em banco de dados
- Visualização de gráficos para análise

## Objetivo do projeto
O sistema foi desenvolvido para auxiliar no controle e redução do desperdício de merenda escolar, permitindo o registro de dados e a geração de relatórios visuais.

## Como executar o projeto
1. Clone o repositório:
2. Abra o projeto na sua IDE.
3. Instale as dependências do projeto no terminal da sua IDE.
4. Configure o banco de dados.
5. Execute o back-end e depois o front-end.
